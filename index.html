30.06.2020 відбулись батьківські збори (в режимі он-лайн) для батьків вступників. Записи нарад можна переглянути у відповідному каналі Steam (доступ з ліцейським акаунтом). В ліцеї працює внутрішній корпоративний простір на основі Microsoft Office 365. Доступ до простору відбувається за допомогою ліцейського акаунту. Всі вступники, слухачі курсів та учні ліцею мають персональний акаунт. Акаунт створюється при подачі заяви на курси або для участі у вступних іспитах. При втраті паролю звертатися до служби підтримки. На початку навчального року також створюються акаунти для батьків ліцеїстів. Батьківські акаунти мають обмежені властивості та доступ і призначені для швидкої комунікації з викладачами та співробітниками та адміністрацією ліцею. Основні навчальні матеріали та посилання для учнів знаходяться на внутрішньому сайті Бібліотеки. На ньому розміщені завдання для учнів для підготовки до нового навчального року на сторінках відповідного предмету.

Відповідно до рішення уряду та МОН під час адаптивного карантину дозволяється проведення вступних іспитів за правилами безпеки для проведення ЗНО. Інфографіка безпеки ЗНО. Вступні іспити будуть проходити протягом 2-х годин. Відстань між учасниками буде не менше 1,5 метрів. Допускаються лише учасники без підвищеної температури та у персональних засобах захисту. Дати проведення на сторінці абітурієнта. Час проведення співбесід та іспитів визначається за персональним графіком, який буде сповіщено на ліцейську електронну пошту учасника напередодні екзамену. Відповідно рішення МОН з 12.03.2020 до 22.05.2020 у закладах освіти запроваджено карантин. Закінчення навчального року планується відповідно наказу МОН України. Для учнів ЛІТ у 4-й чверті: всі заняття відбуваються в он-лайн режимі за розкладом (Розклад та всі посилання на он-лайн матеріали та заходи знаходяться у Бібліотеці). Для учнів підготовчих курсів всі заняття відбуваються за розкладом (Розклад та всі посилання на он-лайн матеріали та заходи знаходяться у Бібліотеці), на ліцейську пошту слухачів курсів будуть надходити повідомлення та уточнення від викладачів).
